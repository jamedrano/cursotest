Lean = Maximizar Valor, Minimizar Desperdicio

Idea Central

Identificar y eliminar sistemáticamente actividades que no agregan valor desde la perspectiva del cliente.

Enfoque en:

Velocidad
Eficiencia
Flujo

Ejemplo Práctico

Si el procesamiento de un reclamo requiere 5 firmas internas cuando solo 2 son realmente necesarias para el cumplimiento/riesgo, las 3 firmas adicionales representan desperdicios potenciales (esperas, procesamiento extra) que el pensamiento Lean cuestionaría.

Proceso con desperdicio: 5 firmas
→
Proceso Lean: 2 firmas

Analogía: Sistema de Tuberías

Imagine un sistema de tuberías obstruido con residuos (desperdicios). Lean se trata de limpiar las tuberías para que el agua (valor) fluya de manera suave y rápida hacia el cliente.

Antes de Lean:

Flujo obstruido por desperdicios - Pase el cursor sobre la tubería para ver qué sucede

Entrada
Salida (Cliente)

Un proceso típico contiene:

5% Valor puro
35% Soporte necesario
60% Desperdicio puro

Lean hace visible lo invisible.
La mayoría de las personas ven el proceso, pero Lean te enseña a ver el desperdicio.
