MSA: ¿Es Confiable Nuestro Sistema de Medición?

"Basura Entra, Basura Sale"

Si la forma en que medimos es defectuosa, nuestros datos y conclusiones también lo serán.

MSA pregunta: ¿Cuánto de la variación observada se debe al proceso mismo y cuánto se debe al sistema de medición?

Conceptos Clave MSA para Yellow Belts:

Exactitud (Sesgo)

¿Está nuestro sistema de medición, en promedio, cerca del valor verdadero?

Bajo
Real
Alto

Ejemplo: ¿Una balanza muestra consistentemente 2kg de más?

Precisión (Variación)

Repetibilidad

¿Puede la misma persona obtener el mismo resultado midiendo el mismo elemento varias veces con el mismo dispositivo?

Consistencia de un evaluador

Reproducibilidad

¿Pueden diferentes personas obtener el mismo resultado midiendo el mismo elemento con el mismo dispositivo?

Consistencia entre evaluadores

Visualizando Exactitud vs Precisión

✓ Ideal

Preciso pero no Exacto

⚠ Sesgo consistente

Exacto pero no Preciso

⚠ Alta variabilidad

Ni Exacto ni Preciso

✗ Problemático

Importancia para Tecniseguros:

  • Asegurar la entrada consistente de datos
  • Interpretación consistente de formularios
  • Informes de sistemas confiables
  • Evaluaciones de riesgo coherentes

Ejemplo Práctico:

"Si dos agentes de Tecniseguros evalúan la 'completitud' de una nueva solicitud de póliza de manera diferente (uno dice 'completa', otro encuentra 3 elementos faltantes), nuestro sistema de medición para 'completitud de solicitud' tiene un problema de reproducibilidad."
