Diagrama de Flujo con Carriles (Swimlanes)

Muestra pasos Y quién (departamento, rol, sistema) es responsable de cada paso.

Excelente para visualizar:

  • Transferencias entre departamentos
  • Procesos entre múltiples funciones
  • Demoras entre "carriles"

Ejemplo: Proceso de reclamos

Cliente
Envía reclamo
Servicio al Cliente
Registra reclamo
Ajustador

Pagos
